# 🎮 Dark Frontier - Mini Shooter

Dark Frontier is a beginner-friendly shooter game project built using multiple programming languages.

## 🚀 Features

* Player Movement (W, A, S, D)
* Shooting System
* Enemy Spawning
* Health System
* Score System
* Game Over Screen
* Python Backend Support
* C++ Damage Calculation Module
* Expandable Architecture

## 🛠 Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Python (Flask)

### Performance Module

* C++

## 📁 Project Structure

```text
MiniShooter/
│
├── index.html
├── style.css
├── script.js
├── server.py
└── damage.cpp
```

## 🎮 Controls

| Key   | Action     |
| ----- | ---------- |
| W     | Move Up    |
| A     | Move Left  |
| S     | Move Down  |
| D     | Move Right |
| Space | Shoot      |

## ⚙️ Setup

### Run Frontend

Open:

```text
index.html
```

in your browser.

### Run Python Server

Install Flask:

```bash
pip install flask
```

Start server:

```bash
python server.py
```

### Compile C++

```bash
g++ damage.cpp -o damage
```

Run:

```bash
./damage
```
