- `style.css`
- `script.js`

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Backend: Python Flask support
- Performance module: C++

## Controls

- Move: W, A, S, D or Arrow Keys
- Boost fire: Space
- Mobile: drag on the screen

If the link does not show the new game yet, upload these updated files to the repository's `main` branch. GitHub Pages is already using this URL and will refresh after the update.
