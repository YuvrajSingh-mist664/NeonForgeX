# NeonForgeX

## Live Browser Game

[Play NeonForgeX](https://yuvrajsingh-mist664.github.io/NeonForgeX/)

NeonForgeX is a browser-ready arcade shooter built with HTML5, CSS3, and JavaScript. The game opens directly from the GitHub Pages link above after these files are uploaded to the repository's `main` branch.

## Features

- Full-screen Canvas gameplay
- Player movement
- Auto-fire combat
- Enemy waves
- Hull health system
- Score and best score
- Desktop and mobile controls
- Python Flask backend support
- C++ performance module placeholder

## Technologies Used

### Frontend

- HTML5
- CSS3
- JavaScript

### Backend

- Python Flask

### Performance Module

- C++

| Input | Action |
| --- | --- |
| W, A, S, D | Move |

| Space | Faster firing |
| Touch / Mouse Drag | Move on screen |

## Run Locally

Open `index.html` in a browser, or run a simple local server:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```
